# idk-ui

Shared design system and component library for the idk website ecosystem.

## Included in v0.1

- Design tokens
- Theme and motion CSS
- Core form and layout components
- Dialog and toast primitives

## Usage

```tsx
import "idk-ui/theme.css";
import "idk-ui/motion.css";
import { Button, Card, PageHeader } from "idk-ui";
```

## Component Docs

For a quick catalog of exported components, usage examples, and grouping by primitives vs patterns, see [COMPONENTS.md](./COMPONENTS.md).

## Demo

Run the interactive demo app:

```bash
npm run demo:dev
```

Build the demo:

```bash
npm run demo:build
```
